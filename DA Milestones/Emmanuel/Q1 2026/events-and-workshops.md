# Community Engagement Report – Q1



As part of the Community Engagement milestone for Q1, I organized and facilitated three community-focused events in Nairobi aimed at strengthening the local Cardano developer ecosystem and encouraging participation in open-source development.

The events organized were:

1. Cardano Hub Nairobi Monthly Meetup  
2. Cardano Hackathon (Workshop)  
3. Intersect Core Repositories Workshop (Repository Walkthrough)

These events were designed to support developers, encourage collaboration, and promote active contributions to the Cardano ecosystem.


# Event 1: Cardano Hub Nairobi Monthly Meetup

## Event Description

The **Cardano Hub Nairobi Monthly Meetup** is a recurring community event focused on maintaining an active developer and builder community within the Cardano ecosystem in Nairobi.

The purpose of the meetup is to provide a space where developers, enthusiasts, and contributors can stay updated on ecosystem developments while collaborating and learning together.

## Activities During the Meetup

The meetup typically includes several segments:

- **Ecosystem Updates**  
  Updates on developments within the Cardano ecosystem, including governance discussions, tooling improvements, and community initiatives.

- **Developer Talk Sessions**  
  Technical sessions delivered by myself focused on Cardano development topics, tools, and opportunities for contribution.

- **Community Planning**  
  Discussions about upcoming initiatives and plans for the Nairobi Cardano Hub.

- **Community Engagement Activities**  
  Interactive sessions including games and quizzes where participants can earn small **ADA rewards** for participation.

These meetups help maintain continuity within the local Cardano developer community and encourage ongoing engagement with ecosystem initiatives.

## Outreach and Participation

The meetup attracted local developers, blockchain enthusiasts, and community members interested in learning about Cardano development.

Participants were able to:

- Learn about current developments in the Cardano ecosystem
- Engage in technical discussions around development tools and practices
- Connect with other builders within the community

# Event 2: Cardano Hackathon (Workshop)

## Event Description

The **Cardano Hackathon** served as the primary workshop event for this quarter and was organized in collaboration with **Blockchain Center Nairobi**.

This collaboration was initiated as part of ongoing outreach efforts to build stronger connections between the Cardano ecosystem and local blockchain communities.

The goal of the hackathon was to encourage developers to actively contribute to **Cardano open-source repositories**, particularly those related to Intersect and the broader ecosystem.

Unlike traditional hackathons focused on building standalone applications, this event was designed to introduce developers to the **open-source contribution process** within the Cardano ecosystem.


## Objectives of the Workshop

The hackathon workshop focused on several goals:

- Introducing developers to Cardano open-source development
- Encouraging contributions to **Intersect core repositories**
- Helping participants understand the GitHub workflow used in open-source projects
- Building familiarity with identifying and solving issues within existing repositories
- Encouraging long-term participation in the Cardano developer ecosystem


## Workshop Structure

The workshop was structured around practical participation and hands-on contribution.

Activities included:

- Introduction to Cardano open-source development
- Overview of the Intersect repositories and contribution guidelines
- Identifying beginner-friendly issues for participants
- Collaborative problem-solving during the hackathon session
- Submitting pull requests addressing identified issues

Participants worked individually or in small groups to review issues and submit improvements to the relevant repositories.


## Contributions and Outcomes

The hackathon resulted in several open-source contributions from participants.

Developers engaged directly with repository issues and submitted pull requests addressing various problems identified within the ecosystem.

# Event 3: Intersect Core Repositories Workshop (Repository Walkthrough)

## Event Description

A technical workshop was organized to provide developers with a **practical walkthrough of Intersect’s core Cardano repositories** and how they work together to operate the Cardano network.

The session focused on helping developers understand the **layered architecture of Cardano**, using the **Cardano Blueprint** as a reference for explaining how protocol components interact.

The workshop included live demonstrations and walkthroughs of several important repositories including:

- `cardano-base`
- `cardano-ledger`
- `ouroboros-network`
- `cardano-node`
- `cardano-cli`
- `cardano-db-sync`

Participants were shown how these repositories connect across the **protocol, networking, and application layers**, giving them a clearer understanding of the overall system architecture.


## Workshop Goals

The session aimed to:

- Provide a clear overview of how the Cardano network operates internally
- Help developers navigate and understand key repositories within the ecosystem
- Demonstrate how the **Cardano Blueprint** maps architecture to implementation
- Lower the barrier for developers interested in contributing to core repositories


## Outcomes

The repository walkthrough helped developers build a clearer mental model of how Cardano components interact and where contributors can begin exploring the codebase.

Participants gained insight into:

- The relationship between specifications and implementation
- How different protocol layers interact
- Entry points for developers interested in contributing to the ecosystem

# Key Observations

Across these events, several insights emerged:

### Interest in Open Source Contribution
Many developers are interested in contributing to blockchain projects but need guidance on how to begin contributing to existing repositories.

### Importance of Community Spaces
Regular meetups help maintain momentum and ensure developers remain engaged with ecosystem developments.

### Value of Hands-On Workshops
Workshops such as hackathons and repository walkthroughs provide practical environments where developers can transition from learning about blockchain technology to actively contributing to it.
